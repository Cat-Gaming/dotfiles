# FAQ

# What is this?
This is my configuration for my dotfiles.

# How do I copy it over to my system?
```
git clone https://github.com/Cat-Gaming/dotfiles
cd dotfiles
cp .zshrc /home/$USER/.zshrc
mkdir ~/.config/awesome
cd awesome
cp *.lua ~/.config/awesome
```
Then you're done!

# Dependencies

